# android-signup

### 🚀 2단계 - 회원가입(뷰)
- 제목 컴포저블 만들기
- 텍스트 필드 컴포저블 만들기
- 버튼 컴포저블 만들기
- 화면 구성하기
- 테스트 코드 작성하기

### 🚀 3단계 - 회원가입(유효성 검사)
- 텍스트 필드 유효성 검사하기
    - 이름 유효성
        - 이름의 글자 수가 2~5자여야 한다
        - 이름에 숫자나 기호가 포함되어 있으면 안 된다
    - 이메일 유효성
    - 비밀번호 유효성
        - 비밀번호가 8~16자여야 한다
        - 비밀번호에 영문과 숫자가 포함되어 있어야 한다
        - 비밀번호와 비밀번호 확인이 일치해야 한다
- 에러 UI 표시하기
- 테스트 코드 작성하기
