# Step 2 - 회원가입(뷰)

## 기능 요구 사항

1. 디자인 시안을 참고하여 회원가입 뷰를 구현한다.

## 프로그래밍 요구 사항
1. ViewModel, Hilt 등은 회원가입 미션에서 활용하지 않는다. 컴포즈 학습에 집중하자.
2. 사용자 입력 및 유효성 검사에 대해서는 이 단계에서 고민하지 않아도 된다.
3. 컴포저블 함수가 너무 많은 일을 하지 않도록 분리하기 위해 노력해 본다.
4. Material3 Button, TextField를 활용한다.
